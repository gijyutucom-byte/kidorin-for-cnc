# きどりん for CNC
## STL to 木取り図作成ツール

**STLファイルから木材の木取り図を自動生成し、CNC加工用Gコードまで一括出力できる、ブラウザで動作するWebアプリです。**

## 概要

**きどりん for CNC** は、STLファイルを読み込むだけで部品を板に自動配置し、ジョイント加工・タブ設定・Gコード生成までをブラウザ上で完結させるツールです。インストール不要で動作します。

---

## 特徴

- **STLを読み込むだけ** で部品を板に自動配置
- **複数枚の板**に横並びで部品を整理でき、板ごとに個別Gコードを生成
- **相欠きジョイント**（凹凸組み手）とTボーン逃げ加工を自動生成
- **タブ設定**で切り抜き時の部品飛び出しを防止
- **Mach2/3形式のGコード**を直接ダウンロード（加工順序は左下→右→上に最適化）
- **Gコードビュワー**でシミュレーション確認（タブ位置・加工時間の表示付き）。Gコード生成前でも起動でき、外部のGコードファイルを読み込んで単独確認することも可能
- **3Dビュワー**でジョイント・Tボーン加工後の形状をSTL/GLB形式でエクスポート可能
- PDF印刷・SVGエクスポートで木取り図を記録・共有
- **板の歩留まり・木目方向警告・部品リスト（BOM）** をリアルタイム表示

## 使い方

### STEP 1　STLファイルを読み込む

- 画面上部の **ファイル選択ボタン** からSTLを選ぶか、表示エリアに **ドラッグ＆ドロップ** します。
- Fusion 360・FreeCAD・TinkerCAD などで作成したSTLが使えます。
- 複数の部品を1つのSTLにまとめた場合（結合STL）、パーツを自動分離して個別に配置します。
- 右上の「**3D表示**」ボタンで、元の3D形状をポップアップウィンドウで確認できます。
  - ポップアップ内の「**STLエクスポート**」「**GLBエクスポート**」ボタンから、表示中の形状（ジョイント・Tボーン加工後も含む）をファイル出力できます。GLBは部品ごとの色情報も保持されます。

### STEP 2　板のサイズを設定する

| 入力項目 | 説明 |
|---|---|
| 板1の長さ (mm) | 材料の長手方向のサイズ |
| 板1の幅 (mm) | 材料の短手方向のサイズ |
| 部品の間隔 (mm) | 部品間の最小すきま（初期値 20mm） |

- 材料が複数枚必要な場合は「**板を追加**」ボタンで増やします（板は横並びに表示）。
- 板を追加後は「**レイアウト初期化**」を押すと部品が再配置されます。

### STEP 3　部品を配置する

| 操作 | 動作 |
|---|---|
| ドラッグ | 部品を移動 |
| 右クリック または ダブルタップ | 90度回転 |
| 板の外をドラッグ | 全体をパン移動 |
| Zoom＋ / Zoom－ | 拡大・縮小 |

- 部品の**長い辺が木目方向（板の長手方向）** と合うように配置してください。合っていない部品は**赤色の警告**が表示されます(合板の場合は対応不要)。
- 画面下部に**板の歩留まり**（材料利用率）と**部品リスト（BOM）** がリアルタイム表示されます。

### STEP 4　ジョイントとTボーンを生成する（CNC加工用）

複数の部品を組み合わせて使う場合に使用します。

| ボタン | 機能 |
|---|---|
| **ジョイント生成** | 交差部分に相欠きジョイント（凹凸組み手）を自動追加 |
| **Tボーン作成** | 内コーナーに円形の逃げ加工（Tボーン）を挿入 |
| **ジョイント＆Tボーン一括生成** | 上記を同時に処理 |
| **STL初期状態に戻す** | ジョイント・Tボーンの変更をすべてリセット |

**設定項目：**

- **ミル径（mm）**：使用するエンドミルの直径（6mm または 12mm）
- **クリアランス（mm）**：ジョイントのはめ合いすきま（初期値 0.2mm）

> ジョイント生成後も部品の位置・角度・タブ設定は保持されます。
> ジョイント・Tボーン加工後の形状は「3D表示」ポップアップからSTL/GLBとしてエクスポートできます。

### STEP 5　タブを設定する

CNCで部品を切り抜く際、最終パスで部品が飛ばないようにタブを設定します。

1. 「**タブ設定**」ボタンを押すとタブモードになります（ボタンが赤く点滅）。
2. 部品の**輪郭線の上をクリック**すると、赤い□マークのタブが追加されます。
3. タブをクリックすると削除できます。
4. もう一度「タブ設定中（完了）」を押すとモードが終了します。

**タブサイズ（残し高さ：6mm）：**

| エンドミル | タブ長さ |
|---|---|
| 6mm | 16mm |
| 12mm | 20mm |

### STEP 6　Gコードを生成する

「**Mach2/3用Gコード生成**」ボタンを押すと、ダイアログで以下を確認・入力して生成します。

| 項目 | 6mm初期値 | 12mm初期値 |
|---|---|---|
| 材料の厚さ (mm) | — | — |
| 1パス目の切削深さ (mm) | 材料厚×0.5625 | — (1パスのみ) |
| 切削送り速度 (mm/min) | 2000 | 1500 |
| 切り込み速度 (mm/min) | 2000 | 1500 |
| スピンドル回転数 (rpm) | 18000 | 17000 |
| 安全高さ (mm) | 7 | 7 |

- 部品は**左下原点から左→右・下→上の順**に加工されます。
- 板が複数ある場合は**板ごとに別ファイル**をダウンロードします（例：`設計名_板1_mach23.txt`）。
- タブが設定されている場合は、タブ位置でZ軸を自動的に残し高さ（6mm）まで持ち上げます。
- 出力形式：**Mach2/3 形式**（G21 / G90 / Exact Stop G61）

### STEP 7　Gコードをビュワーで確認する

「**Gコードビュワー**」ボタンでシミュレーション画面が開きます。Gコードを生成していない状態でも起動でき、その場合は空の状態で開き、後からファイルを読み込んで確認できます。

| 機能 | 説明 |
|---|---|
| G-codeファイル読み込み | 画面内のボタンから、本アプリで生成したものとは別のGコードファイル（.txt/.nc/.tap/.gcode/.cnc）を読み込んで単独でシミュレーション確認できます |
| 3Dプレビュー | 工具パスを早送り・切削・タブ別に色分け表示 |
| 再生コントロール | ×1〜×1000の速度で加工順序をアニメーション再生 |
| スライダー | 任意のステップにジャンプ |
| 加工時間表示 | 総加工時間と経過時間をリアルタイム表示 |
| 板の選択 | 複数板がある場合、セレクターで板を切り替えて確認 |
| 評価レポート | タブ検出・座標範囲・行番号チェックなどを自動判定 |

**凡例（色分け）：**

| 色 | 意味 |
|---|---|
| 水色（薄） | 早送り（G00） |
| オレンジ | 切削 1周目（浅い） |
| 暗赤 | 切削 2周目（深い） |
| 黄色 | タブ区間 |
| 白（枠） | 材料サイズ |

### STEP 8　木取り図を保存・印刷する

| ボタン | 出力 |
|---|---|
| **PDF表示** | A4横置きの木取り図を印刷・PDF保存（タブマーク除外） |
| **SVG保存** | ベクター形式のSVGファイルをダウンロード（タブマーク除外） |

## 技術仕様

### STL読込・部品分離

- バイナリ・テキスト両形式のSTLに対応
- 共有エッジの隣接グラフをたどり、連結成分ごとに部品を自動分離
- XY・XZ・YZ の3方向に投影し、最大面積の投影を2D輪郭として使用

### Gコード仕様

- **コントローラ形式**：Mach2/3（G21 mm単位 / G90 絶対座標 / G61 Exact Stop）
- **工具半径補正**：ツールパスをエンドミル半径分だけ外側にオフセット（Miter接合）
- **2パス切削**：6mm エンドミルは1パス目（荒削り）＋2パス目（仕上げ）の2段階
- **タブ制御**：TAB UP/DOWN コメント付きGコードでMach2/3のスクリプトと連携可能
- **行番号**：N番号を10刻みで自動付番

### ジョイント生成アルゴリズム

- 部品のバウンディングボックスの交差を検出
- 交差方向（最大延長軸）に相欠きの凹凸を交互に配置
- 歯の長さが 100〜150mm になるよう奇数分割数を自動選択
- CSG（Constructive Solid Geometry）演算で3Dメッシュを直接加工

### 3Dモデルエクスポート（3D表示ポップアップ）

- **STL**：表示中の全メッシュ（ジョイント・Tボーン加工後の形状を含む）を結合し、バイナリSTL形式で出力。外部ライブラリ不要で頂点座標・面法線を直接生成
- **GLB**：glTF 2.0バイナリ形式で出力。部品ごとのマテリアル（色・粗さ・金属度）情報を保持。外部ライブラリ不要でJSON＋バイナリチャンクを直接構築

### Gコードビュワーの評価レポート

- TAB UP/DOWNコメントの検出・ペア確認、実タブ（最終周回）の自動判定
- 周回深さ・タブZ値・タブ高さ・タブ幅の検証（標準幅から外れる短いタブは警告表示のみで、総合判定の不合格条件には含めない）
- 座標範囲が材料サイズ内に収まっているかの確認、M30命令・%マーカー・行番号連続性のチェック
